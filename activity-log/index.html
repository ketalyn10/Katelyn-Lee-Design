ACTIVITY LOG
2026
ACTIVITY LOG is a visualization-focused extension of KATELYN LEE IS ONLINE, translating six months of browser history into a poster and a set of supporting visual artifacts. Through scale, pattern, and comparison, the work reframes browsing data as something both readable and personally recognizable.
Design
The visual system uses indexing, repetition, and categorization to transform personal data into a structured publication series across 72 unique topics. All visualizations were generated using p5.js, using the browser as both design and printing tool.
Outcome
  • ACTIVITY LOG, a poster translating selected data into a visual format
  • Supporting visualizations expanding on patterns of time, frequency, and browsing behavior
  • A focused extension of the larger thesis system, centered on interpretation
