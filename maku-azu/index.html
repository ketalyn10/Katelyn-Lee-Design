The World Within: Maku Azu
2025
Custom glyph set developed as an extension of the typographic identity for The World Within: Maku Azu, a 2024 exhibition presented during the Venice Biennale and curated by Talia Kirsh. This project was initiated independently following the exhibition, exploring how the existing typographic system could be expanded into a more complete and scalable set of characters.
Design
The work focused on extending an existing glyph system while maintaining consistency in form, proportion, and visual language. Each new character was developed by referencing established shapes and structural rules, ensuring the expanded set remained cohesive and legible.
Outcome
  • Expanded glyph set/alphabet
  • Consistent extension of an existing typographic system
