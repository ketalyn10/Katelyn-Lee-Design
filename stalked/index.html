Have You Been Stalked by Katelyn Lee?
2026
Have You Been Stalked by Katelyn Lee? is a web-based extension of KATELYN LEE IS ONLINE that reframes personal browsing data through search. Users can look themselves up to see if and how they appear within my dataset, turning passive records into an interactive, confrontational experience.
Design
The interface mirrors a search engine, using input and indexed results to query a personal dataset instead of the web.
Outcome
  • An interactive website that allows users to search themselves within a personal dataset
  • A recontextualization of browser history as a searchable, relational system
  • A participatory extension of the thesis, shifting from observation to interaction
